CSS 위치 속성 예제

1. static (기본)

static

문서의 기본 흐름에 따라 배치, top/left 적용 안됨

2. relative

relative

원래 자리에서 top:20px; left:30px; 만큼 이동

3. absolute

absolute

가장 가까운 position: relative 조상을 기준으로 위치

4. fixed

fixed

브라우저 화면을 기준으로 고정 (스크롤해도 움직이지 않음)

5. sticky

sticky
스크롤을 내려보세요!

sticky 박스는 스크롤할 때 원래 위치에서 움직이다가,
top:0에 도달하면 고정됩니다.
